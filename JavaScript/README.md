# JavaScript 面试题

本目录包含 JavaScript 相关的高频面试题，覆盖 JavaScript 核心概念、ES6+新特性、异步编程、原型链、闭包、事件循环等核心知识点。

## 📚 目录内容

- **基础概念** - JavaScript 数据类型、作用域、执行上下文等
- **ES6+特性** - let/const、箭头函数、解构赋值、Promise、async/await等
- **异步编程** - 回调地狱、Promise链、async/await、事件循环
- **核心机制** - 原型链、闭包、this指向、作用域链
- **手写实现** - 常见JavaScript函数的手写实现
- **性能优化** - 内存泄漏、性能调优、代码优化

## 🎯 学习建议

### 重点掌握
1. **原型链和继承** - 面试必考，需要理解透彻
2. **异步编程** - Promise、async/await、事件循环是高频点
3. **闭包和this** - 基础但易错，需要反复练习
4. **ES6+特性** - 现代开发必备，要能熟练使用

### 刷题建议
1. 先理解原理，再记忆答案
2. 对于手写题，一定要亲自实现
3. 掌握一个知识点后，尝试从多个角度提问自己
4. 定期复习，形成知识体系
