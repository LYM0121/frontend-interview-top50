## 📚 题库分类

### 1. [HTML/CSS](/HTML-CSS/) (10题)
- 布局、响应式、动画、性能优化等
- 大厂高频考题整理

### 2. [JavaScript](/JavaScript/) (15题)
- 基础概念、ES6+特性、异步编程
- 原型链、闭包、事件循环、手写函数

### 3. [React](/React/) (10题)
- React 核心概念、Hooks、性能优化
- 生命周期、虚拟DOM、状态管理、React 18新特性

### 4. [Vue](/Vue/) (8题)
- Vue 2/3 核心概念、响应式原理
- Composition API、Vite、Vue Router、Vuex/Pinia

### 5. [浏览器与网络](/Browser-Network/) (7题)
- 渲染机制、性能优化、缓存策略
- HTTP/HTTPS、跨域、安全、WebSocket

### 6. [数据结构与算法](/Data-Structures/) (5题)
- 常考数据结构（数组、链表、树）
- 排序算法、字符串处理、前端特色算法题

### 7. [项目与场景题](/Projects/) (5题)
- 项目经验、系统设计、性能优化方案
- 实际场景解决方案设计

## 📖 使用建议

### 推荐学习路径
1. **基础优先**：HTML/CSS → JavaScript → 框架 → 浏览器网络
2. **重点突破**：针对薄弱环节专项训练
3. **实战演练**：项目与场景题提升综合能力
4. **模拟面试**：按模块自测，记录错题

### 学习建议
- ✅ **理解原理**：不只记答案，要理解背后的原理
- ✅ **手写代码**：面试常考手写实现，务必亲自写一遍
- ✅ **总结归纳**：按主题整理笔记，形成知识体系
- ✅ **实践应用**：思考如何将答案应用到实际项目中

## 🤝 贡献指南

欢迎提交 Issue 或 Pull Request！

### 如何贡献：
1. **提交新题目**：建议包含题目、答案、考察点、难度等级
2. **修正错误**：发现错误请及时提 Issue 或直接提交 PR

- 使用清晰的中文描述
- 保持代码格式一致性
- 添加合适的标签（如难度、公司、技术方向）
