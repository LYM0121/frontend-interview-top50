## 🏢 大厂 React 面试特点

### 腾讯面试重点
- **React Hooks 深度**：自定义 Hook 设计模式
- **性能优化实践**：大数据量列表渲染优化方案
- **工程化能力**：React 项目架构设计

### 阿里字节面试重点
- **源码理解**：React Diff 算法、Fiber 架构
- **状态管理架构**：Redux 替代方案对比
- **设计模式**：组合模式、渲染 Props、HOC

### 快手美团面试重点
- **项目实战**：React 项目性能问题排查
- **团队协作**：组件库设计和维护
- **新技术跟进**：React 18 新特性应用
