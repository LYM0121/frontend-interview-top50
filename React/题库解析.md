## 1.1 什么是React？有什么特点？

### 问题
什么是React？React的主要特点是什么？

### 参考答案
**React**：由 Facebook 开发的用于构建用户界面的 JavaScript 库。

**主要特点**：

1. **组件化**：UI 拆分为独立、可复用的组件
2. **声明式编程**：描述 UI 应该是什么样
3. **虚拟 DOM**：在内存中维护 DOM 副本
4. **单向数据流**：数据从父组件流向子组件
5. **高效**：通过 Diff 算法最小化 DOM 操作

### 参考答案

### 问题

### 参考答案


## 2.8 详细解释所有生命周期方法

### 问题

### 参考答案

## 3.16 如何自定义 Hook？

### 问题
如何自定义 Hook？

### 参考答案
自定义 Hook 是复用组件逻辑的函数，函数名以 "use" 开头。

```javascript
function useCounter(initialValue = 0) {
  const [count, setCount] = useState(initialValue);
  
  const increment = useCallback(() => setCount(c => c + 1), []);
  const decrement = useCallback(() => setCount(c => c - 1), []);
  
  return { count, increment, decrement };
}
```

## 4.21 如何在组件中连接 Redux Store？

### 问题
如何在 React 组件中连接 Redux Store？

### 参考答案

```javascript
import { useSelector, useDispatch } from 'react-redux';

function Counter() {
  const count = useSelector(state => state.count);

  return <button onClick={() => dispatch({ type: 'INCREMENT' })}>{count}</button>;
}
```

## 6.27 React 高阶组件（HOC）是什么？

### 问题
什么是 React 高阶组件（HOC）？

### 参考答案
高阶组件（HOC）是接受组件并返回新组件的函数。

```javascript
function withLoading(WrappedComponent) {
  return function(props) {
    if (props.loading) {
      return <div>Loading...</div>;
    }
    return <WrappedComponent {...props} />;
  };
}
```
