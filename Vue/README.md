# Vue 面试题

本目录包含 Vue 2/3 相关的高频面试题，覆盖 Vue 核心概念、响应式原理、组件通信、Vue Router、Vuex/Pinia、Vue 3 新特性等核心知识点。

## 📚 目录内容

- **Vue 基础** - 核心概念、指令、生命周期
- **Vue 2 响应式** - 数据响应式原理、Observer/Dep/Watcher
- **Vue 3 新特性** - Composition API、Proxy响应式、新生命周期
- **组件系统** - 组件通信、插槽、动态组件
- **状态管理** - Vuex、Pinia 原理和使用
- **路由管理** - Vue Router 原理和高级用法
- **性能优化** - Vue 应用性能优化策略
- **Vite 与构建** - Vite 原理和项目构建

## 🎯 学习建议

### Vue 2 → Vue 3 过渡
1. **先扎实 Vue 2 基础**：生命周期、响应式原理、Vuex
2. **学习 Vue 3 新特性**：逐步掌握 Composition API
3. **对比理解差异**：理解两代框架的设计思路变化
4. **实战项目过渡**：在项目中逐步引入 Vue 3 特性

### 重点掌握
1. **响应式原理**：深入理解 Vue 2/3 响应式实现差异
2. **组件通信**：掌握多种通信方式及适用场景
3. **状态管理**：理解 Vuex/Pinia 的设计思想
4. **性能优化**：掌握常见的优化手段和排查方法
