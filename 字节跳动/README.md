# 字节跳动前端面试专题

本目录专门整理字节跳动（ByteDance）前端开发岗位的高频面试题目、面试经验及技术深度问题。包含抖音、头条、TikTok、飞书等业务线的真实面试题目。

## 📊 字节跳动前端面试特点

### **技术深度要求高**
- **原理理解**：不满足于API使用，要求理解底层实现
- **手写源码**：常见工具、核心功能的实现
- **工程视野**：不只是写代码，还要有架构和工程化思维

### **面试流程典型**
```
一面（技术面） → 二面（技术面） → 三面（技术面/交叉面） → HR面
```
- **一面**：基础八股 + 简单算法 + 项目深度
- **二面**：技术深度 + 系统设计 + 复杂算法  
- **三面**：工程视野 + 架构设计 + 技术视野

### **业务线侧重**
- **抖音/头条**：性能优化、用户体验、复杂交互
- **TikTok**：跨文化、国际化、大流量场景
- **飞书**：企业级应用、协同编辑、实时通信
- **Coze/火山引擎**：AI前端、低代码、智能化

## 🔥 高频面试题目汇总

> 基于真实面试记录整理，持续更新中...

### **1. 一面高频题（基础知识）**
1. **HTTP/HTTPS**：HTTPS加密原理、数字证书、中间人攻击
2. **浏览器**：渲染流程、重排重绘、EventLoop
3. **JavaScript**：闭包、原型链、this指向、手写Promise
4. **React/Vue**：Diff算法、Fiber架构、响应式原理
5. **CSS**：盒模型、Flex布局、BFC、性能优化

### **2. 二面深度题（技术深度）**
6. **React Fiber**：调度机制、时间切片、并发模式
7. **Vue3 vs Vue2**：响应式重写、Composition API、性能提升
8. **前端工程化**：微前端、模块联邦、构建优化
9. **性能监控**：APM系统设计、性能指标采集
10. **代码质量**：如何重构历史遗留代码、保证质量

### **3. 三面系统题（工程视野）**
11. **架构设计**：设计一个前端监控平台
12. **性能优化**：首屏加载从4秒优化到1秒的方案
13. **团队协作**：如何推动技术方案在团队落地
14. **技术选型**：Vue vs React vs Svelte vs Solid 对比
15. **AI+前端**：AI Coding应用、智能化前端实践

### **4. 编程题专项**
16. **防抖/节流**的实现及其应用场景
17. **EventEmitter**事件系统的完整实现
18. **useDataSelector**（Vue3+TS实现复杂状态管理）
19. **嵌套数组转HTML列表**（DOM操作考察）
20. **请求防重**：并发请求参数对应问题

```

✅ 源码阅读（React/Vue核心部分）

✅ 设计能力（系统设计、架构思维）

✅ 压力测试（高强度连续面试练习）
```

## ⚡ 快速启动

### **立即开始练习**
1. 从「一面高频题」开始，确保基础牢固
2. 逐个攻克「编程题专项」，提升实战能力
3. 深入研究「项目挖掘题」，准备深度项目阐述

- 找朋友进行1小时模拟面试

- 录音复盘，找出改进点

- 字节面试注重深度而非广度
- 不会的问题可以思考过程，展示解题思路
- 自信表达，展现学习能力和成长潜力
